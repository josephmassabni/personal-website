I do this in my free time!

What started as using a drag-and-drop software to attempt making video games in middle school evolved to a lifelong passion for programming my ideas into reality.

Today, I have been focusing on JavaScript, HTML, and CSS to channel my curiosity into the process of web development.

Unfortunately, I do not have as much time as I would like to focus on this due to school, work, and other committments.

All projects shown were made solely as learning experiences in order to develop my programming skills.





NDA Robotics Website

I was tasked with creating a website for the NDA Robotics program in order to compete in the Dell Robotics Website Challenge. This was the first large project that I created and allowed me to teach myself the basics of web development.

Learn More Check it out!

I always knew that I wanted to create websites from scratch, but I never knew how. One day, however, I learned that I could create a website for my robotics team in hopes of qualifying for Worlds. Although we did not win an award, I am very happy I embraced the challenge. I was able to learn the basics of HTML and the Bootstrap framework. By the end, I also gained a basic knowledge of CSS and used it to a limited extent on some aspects of the website. The biggest challenges I faced were trying to do way too much at first- I realized that I would have to start simple and work from there! That was a very humbling realization to have. But, I was able to incorporate some of the complex aspects that I planned into my website. In short, I attribute this project to my interest in web development!



Hidden Paws Network

While helping a group of people clean their elderly neighbor's house, I was asked to create a website for a local nonprofit, Hidden Paws Network, while still making it easy for them to post updates. Finally, I was able to use my talents to help the community!

Learn More Check it out!

Hidden Paws Network needed a website to showcase their mission, actions, and requests. They needed a website very quickly in order to keep their work going strong. In order to make it easy for them to post new content if needed, I decided to continue using the Wordpress software that they were previously using. Going from custom programming to a webpage maker seemed like a huge step backwards at first, but eventually I was able to learn to appreciate Wordpress and its ability to allow any organization to create a website quickly and effectively. I was still able to include some custom HTML that allowed this project to still be unique. Most of all, I was happy that I could use my web development knowledge to help a meaningful cause.



My Personal Website

With my newfound time during the summer, I was able to pursue my interest in the idea of developing a personal website. I tried to make this website as interactive and interesting to view as possible, all while learning a lot along the way.

After learning the basics of HTML and the Bootstrap framework, I wanted to complete a project that would allow me to focus more on CSS. I learned so much more about animating elements to make them seem much more interactive and visually appealing. To me, this website felt much more professional than anything I have created in the past. In addition, I focused on making the website responsive on all devices, something I struggled with on my previous projects. I feel that this is the most polished website I have created so far!



Students for Financial Literacy

I created an organization called Students for Financial Literacy, or SFFL for short. We hope to make a difference in the world by providing financial literacy education to all, and I put my web development skills to the test as a result.

Learn More Check it out!

I noticed that there was a lack of financial literacy education at my high schools and schools throughout the country. I created SFFL to solve this issue. In order to do this, I needed to make a website quickly and effectively so I would be able to get started as soon as possible. This was the first website I developed that I had a due date for, and I am happy to say that I was able to make this from scratch in only 3 days, including design and uploading to the server. I feel that my prior projects allowed me to do this so quickly and I am happy with the result. Even more important to me, I was able to apply my skills in order to make a difference in the world!
